> *当职场怀才不遇，遇上创意摸鱼。*

欢迎来到 **无聊宝典** —— 一个塞满杀时间小玩意儿的数字阁楼，全是我在公司浪费我才华的时候捣鼓出来的。
与其再开一场本可以发邮件解决的会议，我选择把闲不住的双手用在真正让我开心的事情上。

## 📜 里面都有啥？

这个仓库收集了一堆没啥用但有趣的小网页——每一个都诞生于极度无聊的时刻，以及一种“哪怕随便做点什么都行”的冲动。

| 项目                        | 简介                                                                                                 |
| --------------------------- | ---------------------------------------------------------------------------------------------------- |
| **🕒 时钟**           | 一个简单干净的时钟。因为时间是我上班时唯一多到用不完的东西。                                         |
| **⏱️ 秒表**         | 用来测量老板回邮件需要多久。（剧透：永远比这个秒表计的时间长。）                                     |
| **💬 自制定聊天界面** | 一个自制的聊天界面，可以添加指定的文本和语音消息，是用来录视频用的原型——毕竟连假聊天也需要设计感。 |
| **💣 无需猜的扫雷**   | 一个*永远*不用靠猜的扫雷变种。每一步都有逻辑可循——不像我的职业生涯。                             |

## 🚀 这玩意儿为什么存在？

老板付我钱，是为了让我“在线待命”，但不一定是为了让我动脑子。
所以我决定把创造力夺回来，一次一个无用项目。
这就是 **被浪费的才华** 转化为 **有趣的产出** 的见证——纪念那些本可以用来做点“有意义”的事、最终却落在这里的时间。

## 🛠️ 如何运行这些杰作

所有项目都是静态的 HTML/CSS/JavaScript 页面。
你可以在本地直接打开任意文件夹里的 `index.html`，也可以部署到任何支持静态文件的地方（GitHub Pages、Netlify 等）。

bash

```
git clone https://github.com/yourusername/The-Boredom-Codex.git
cd The-Boredom-Codex
# 打开任意项目文件夹，启动 index.html
```

## 🧰 构建工具

* 纯 HTML、CSS 和 JavaScript（不用框架——咱们保持简单）
* 大量的阴阳怪气
* \~\~咖啡因\~\~（不喝牛马饮品）
