> *Where corporate underemployment meets creative escapism.*

Welcome to **The Boredom Codex** — a digital attic filled with time-killers I built while my company was busy wasting my talent.
Instead of sitting through another meeting that could have been an email, I chose to channel my idle hands into things that actually amuse me.

## 📜 What's inside?

This repository is a collection of small, pointless (but fun) web pages — each born from a moment of extreme boredom and a desire to build something, anything, with my own two hands.

| Project                            | Description                                                                                                                                                               |
| ---------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **🕒 Clock**                 | A simple, clean clock. Because time is the only thing I have too much of at work.                                                                                         |
| **⏱️ Stopwatch**           | Measures how long it takes for my boss to reply to an email. (Spoiler: it's always longer than this.)                                                                     |
| **💬 Custom Chat UI**        | A self-made chat interface that can display custom text and voice messages. Built as a prototype for recording videos — because even fake conversations need a designer. |
| **💣 No‑Guess Minesweeper** | A Minesweeper clone that*never* forces you to guess. Every move is logically solvable — unlike my career path.                                                         |

## 🚀 Why does this exist?

My employer pays me to be available, but not necessarily to use my brain.
So I decided to reclaim my creativity, one useless project at a time.
This is **wasted talent** turned into **playful productivity** — a monument to all the hours that could have been spent on something "meaningful" but ended up here instead.

## 🛠️ How to run these masterpieces

All projects are static HTML/CSS/JavaScript pages.
You can run them locally by opening any `index.html` file in your browser, or deploy them anywhere that serves static files (GitHub Pages, Netlify, etc.).

bash

```
git clone https://github.com/yourusername/The-Boredom-Codex.git
cd The-Boredom-Codex
# open any project folder and launch index.html
```
